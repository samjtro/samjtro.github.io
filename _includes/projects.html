go-tda

  • Golang API Integration
  • Data Manipulation
  • Custom Struct Manipulation
  • Go Module Publishing & More...

go-tda is a TD-Ameritrade API Hook created by me. It returns valuable information on specific Securities, Price History + Real Time Quotes, Option Chains for a variety of strategies, Account History & more. It also implements the API to place, remove and modify trades, implementing both Data & Trading capabilities in the same API. The end goal is for it to be used in further Algoritrhmic Trading models.
